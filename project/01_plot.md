# Plot

## Core Conflict
- 主人公: サトー（本名: 佐藤一 / さとうはじめ）。本文中の表記はサトー。45歳、バツイチ独身、子なし。東京で静かに暮らす中年男性
- 欲望: 両親の墓へきちんと手を合わせ、誰も住まなくなった実家で一晩だけでも穏やかに過ごしたい
- 障害: 本人の知らないところで、国家防衛の極秘システムを起動できる最後の鍵として監視され、帰省日程まで把握されたうえで孤立しやすい地方路線で暗殺者に狙われる
- 失敗コスト: サトーが死ぬか敵対勢力に確保されれば国家防衛の切り札が非常時に使えなくなるだけでなく、彼自身が大事にしている帰る場所の記憶まで暴力で汚される

## Arc Overview
### Beginning
東京の朝。サトーは都内の東央駅から、架空の山岳特急「みねかすみ」に乗り込む。旅の気分に浸る本人のすぐ近くで、セラは車内販売員に化け、駅と車内で仕掛けられる事故偽装の暗殺を、相手を眠らせたり動けなくしたりしながら静かに封じていく。

### Middle
昼。終点の霧伏駅へ着いたサトーは、花を買い、山あいの静輪寺墓苑へ向かう。墓前で両親へ近況を語る時間はこの物語で最も静かな核であり、その静けさを守るためにセラは寺や山道に潜む狙撃と盗聴を徹底して無力化する。

### Ending
夕方から翌朝。サトーは杉名地区に残る実家へ帰り、風を通し、湯を沸かし、古い家の静けさに身を落ち着ける。誰も住まない家を荒らそうとする最後の襲撃もセラが家の外で眠らせて無力化し、サトーは「帰ってきてよかった」と思いながら一泊の夜と朝を静かに終える。

## Hidden Truth
- サトーはかつて、国家防衛の切り札となる極秘システムの開発責任者だった
- 開発中、本来は試験専用だったサトーの全身生体認証データがテスト環境に登録され、そのまま事故で本番機へ残留した
- その防衛システムは最終起動時にも登録済み本人の全身生体認証を要求するため、正式運用後もサトーが事実上の最後の鍵になっている
- 政府内部では不祥事の発覚を恐れてこの事実を隠蔽し、「非常時にだけ密かにサトーを呼べばよい」と問題を先送りしてきた
- その秘密は第三国へ漏洩しており、敵対勢力は防衛システムを麻痺させるためサトーの殺害または確保を狙っている
- サトー自身は気ままに日常を過ごしているのだが、自分が国家防衛の鍵になっているとは知らない
