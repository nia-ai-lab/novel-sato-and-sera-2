# Chapter Outline

## Release Strategy
- 1回で見せる分量: 帰省編として3章6話12節を同規模でまとめ、章ごとに画像とWeb確認ができる状態にする
- 章あたりの体験: 旅路、墓参り、実家滞在で空気を変えつつ、サトーの優雅さとセラの無音護衛を並走させる
- 話あたりの役割: 1話ごとに移動、滞在、祈り、休息などひとつの行為のまとまりを扱う
- 節あたりの役割: 1節は必ず1事件だけを扱い、その中を `サトーパート` と `セラパート` に分けて同時刻の出来事を重ねる
- 画像生成の最小単位: 節内パート

## Initial Package
- 初回承認前に確定するもの:
  - 全体プロット
  - 主要キャラクター
  - 主要キャラクター画像
  - 背景イメージ

## Chapters

### 第1章
- 章テーマ: 山へ向かう朝の移動と、車内に潜む気づかれない護衛
- 到達点: サトーの帰省気分と、売り子に化けたセラの面白さを定着させる
- ユーザ確認ポイント: 列車旅の空気、車内販売員セラの見せ方、東京から山へ抜ける映像感

#### 第1話
- 話テーマ: 東央駅を発つ
- 到達点: 帰省の目的と、駅時点ですでに始まっている護衛を提示する

- セラパート: 清掃係に紛れて発車標の制御盤工作を止め、ホームの秩序を守る

- セラパート: 観光客風の襲撃者が持ち込んだ器具を無力化し、車内の静けさを保つ
