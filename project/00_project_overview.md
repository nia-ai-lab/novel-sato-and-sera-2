# Project Overview

## Logline
45歳バツイチ独身のサトーこと佐藤一は、両親の墓参りのため久しぶりに一泊で山の故郷へ帰る。本人は静かな帰省を楽しんでいるだけだが、その裏では猫耳メイドの凄腕護衛セラが、車内販売員や土地の気配に紛れながら暗殺者たちを無音で捌き続ける。

## Audience
- 想定読者: 旅情のある日常描写と、気づかれない護衛アクションの落差を楽しみたい読者
- 読後感: 「帰省の静けさがここまで守られていたのか」という可笑しさと、サトーの慎ましい優雅さが残る
- 連載ペース: 章ごとに確認しながら進める創読形式

## Creative Intent
- この作品でやりたいこと: 東京から山の田舎へ向かう移動の気分、墓参りのしんとした感情、空き家になった実家のやわらかい休息を、裏の護衛劇と同時進行で描く
- 絶対に入れたい要素: 架空の電車名と地名、特急車内で売り子に化けるセラ、静かな墓参り、古民家風だが近代の手触りが残る実家、一泊の余韻
- 避けたい要素: 露悪的な郷愁、陰惨すぎる墓地描写、サトーが危機を察知して超人的に動く展開、セラの正体が雑に露見する展開

## Product Intent
- 創読としてどんな体験にしたいか: 読者は帰省旅の手触りを味わいながら、節ごとに「いま裏で何が処理されたのか」を静かに発見していく
- 画像生成の方針: 主要キャラクター画像は既存を継承し、背景は「山岳特急」「山あいの墓苑」「古民家風の実家」を新規トーンとしてそろえる。節画像は全12節ぶん更新する
- モバイル Web での見せ方: 車窓、石段、縁側など奥行きのある画面を多めにして、旅の抜け感と護衛の緊張感を同居させる

## Approval Context
- 既存作品から継承するもの: サトーとセラの基本設定、キャラクター画像、現代スパイ日常コメディとしての世界観
- 今回置き換えるもの: タイトル、プロット、章立て、本文、背景イメージ、節画像、Web公開内容
- 本文開始の前提: ユーザが既存作の差し替えを許可済みで、世界観とキャラクター踏襲の条件が明示されている

## Current Focus
- 今回進める範囲: 帰省編の設計同期、3章6話12節の本文差し替え、画像定義更新、Webビルド
- 未確定事項: 地方警備側の正式部署名、サトーの実家を今後どう維持するか、帰路を次話で描くかどうか
- 次に決めること: 生成画像の仕上がり確認と、必要なら本文の微調整
