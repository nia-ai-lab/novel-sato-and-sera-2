# Worldbuilding

## Setting Summary
現代日本によく似た世界。東京の都心から、架空の山岳路線と小さな町へ移動できる距離感のなかで、一般市民の帰省風景と国家規模の保全任務が同時進行している。今回の主舞台は、都内の東央駅、青嶺本線の特急「みねかすみ」、終点の霧伏駅、静輪寺墓苑、霧伏町杉名地区の実家。

## Key Background Concepts
- 初期確認で見せる背景 01: 朝の特急ホームと、山へ向かう車窓の静かな緊張
- 初期確認で見せる背景 02: 山あいの寺と墓苑、石段、春の薄い光
- 初期確認で見せる背景 03: 古民家風だが昭和後期以降の手触りが残る実家。木の梁、すりガラス、アルミサッシ、縁側、庭木

## Rules
- 魔法・技術: 超常要素はない。すべて現代的な諜報、監視、暗器、事故偽装、通信妨害で成立する
- 社会: 一般の帰省客や町の住民は平穏な日常を送っているが、国家安全保障の最深部ではサトーを守る秘匿任務が続いている
- 地理: 東京から特急で数時間の山間部。駅前は小さいが生活感があり、寺や墓苑や実家までは徒歩と短い車移動で行ける
- 建築: 実家は「古民家風」だが、極端な茅葺きではなく、柱や建具に木の古さがありつつ設備は比較的近代寄り
- 禁則: 架空の地名と列車名は使うが、観光ポスターのように誇張しすぎない。生活感と静けさを優先する

## Sensory Notes
- 色: 朝の駅は銀と淡い群青、墓苑は白い石と若草色、実家は飴色の木と夕方の琥珀
- 質感: 車窓ガラス、古い座席モケット、線香の灰、墓石の冷たさ、畳、綿の座布団、磨り減った廊下
- 音: 発車ベル、車輪の継ぎ目、山の風、寺の鐘、古い鍵、やかんの湯気、雨戸のこすれる音
- 匂い: 駅弁の出汁、車内コーヒー、線香、湿った苔、古い木の家、ほうじ茶

## Open Questions
- 地方側協力者の正式な立場
- サトーが実家を今後どう維持するか
- 次回、帰路や町の別の場所を描くかどうか
