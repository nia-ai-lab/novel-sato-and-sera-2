# Characters

## Cast Rules
- 各キャラクターは「役割」「内面」「外見」「口調」「変化」を必ず持たせる

## Main Cast

### character_id: sato
- 名前: 佐藤一（さとうはじめ）。本文中の表記はサトー
- 役割: 表の主人公。東京在住の中年男性で、帰省そのものを丁寧に味わう人
- 年齢・立場: 45歳。バツイチ独身、子なし。民間のシステム監査と資料整理の仕事を続けている
- 秘匿経歴: かつて国家防衛の極秘システム開発責任者だったが、現在は完全に現場を離れた一般人として暮らしている。本人は過去の案件を終わった仕事として扱っている
- 外見: 身長175cm。やや長身で細マッチョ。黒髪の長髪を普段は後ろでポニーテールに結ぶ。口ひげと顎ひげを整え、年齢相応の渋さとワイルドさがある
- 服装・持ち物: アメカジスタイルを好み、デニム、ワークシャツ、レザーブーツ、ミリタリージャケット、古い腕時計、帆布のトートバッグが似合う。帰省時は文庫本、小さな水筒、墓参り用の封筒などを整えて持つ
- 性格: 温厚、几帳面、観察眼はあるが危険への想像力だけが妙に低い
- 話し方: 落ち着いていて丁寧。少し自虐的
- 欲望: きちんと墓参りを済ませ、誰も住まない実家の空気を一晩だけでも落ち着いて味わうこと
- 恐れ: 先祖や家の記憶を雑に扱うこと、大きな揉め事、若いころの失敗を思い出すこと
- 変化: 帰省の終わりに「帰る場所はまだ自分の中に残っていた」と感じるが、読者にはその時間が徹底して守られていた結果だとわかる
- 匂わせ運用: 本文では国家防衛システムや生体認証の事実を直接説明しない。元政府案件に関わっていた気配、妙に身元確認へ慣れている所作、過去を濁す受け答えなどで一貫して示唆する
- 画像プロンプト要約: 長髪ポニーテールと口顎ひげを持つ、ワイルド寄りの45歳男性。細マッチョで、アメカジが自然に似合う。旅先でも生活感と渋さが共存する

### character_id: sera
- 名前: セラ
- 役割: 裏の主人公。猫耳メイド姿を基調とする、国家レベルの凄腕ボディーガード
- 年齢・立場: 見た目は20代半ば。内閣系秘匿警護部門の実働担当
- 任務前提: サトー本人には極秘システムの真相を知らせず、防衛上の最後の鍵として秘匿護衛する。任務の本質は暗殺阻止だけでなく、敵対勢力による拉致や追跡からの遮断にもある
- 外見: 身長160cm。しなやかで無駄のない体つき。銀灰色の髪、鋭い金色の目。猫耳カチューシャに見える感覚補助装備を持つ
- 服装・持ち物: 白黒のクラシックメイド服を正統派の意匠で着こなす。必要に応じて上から車内販売員のボレロやエプロンなどを重ね、一般客からは制服の一部に見えるよう処理する。暗器や細いワイヤー、投擲針、記録阻害用の小型装置を仕込む
- 性格: 忠実、冷静、毒舌気味。サトーのささやかな幸福を本気で守っている
- 話し方: 短く要点だけ。独り言の時だけ少し感情が漏れる
- 欲望: サトーに一切気づかれず、帰省を無事に完遂させること
- 恐れ: 自分の不始末で、墓参りや実家の静けさを壊すこと
- 変化: 任務を超えて、サトーが大切にしている「帰る行為」そのものへ敬意を深めている
- 画像プロンプト要約: 正統派クラシックメイド服を最優先しつつ、必要なら車内販売員や土地のサービス要員へ自然に擬態できる。可愛さよりも静かな緊張感と手際の良さを優先する
